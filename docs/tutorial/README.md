## 💡 학습 팁

### "쉬운 버전"을 지향했어요

박교수님 말씀처럼 **쉽게** 가르치는 게 목표. 그래서:

- ✅ 명령어는 복붙 가능하게 1줄씩 분리
- ✅ 각 명령의 "기대 출력" 을 같이 적음
- ✅ 학생들이 자주 막히는 데를 미리 안내 (⚠️ 박스)
- ✅ 비유로 개념 설명 (장바구니 = git add 등)
- ❌ 영어 전문 용어 X (필요하면 풀어 씀)

### CMD 우선, GUI 보조

학생들 대부분 PowerShell 안 써봐서 **CMD 위주**. 필요하면 파일 탐색기.

### 막힘 = 자료

각 챕터 끝의 **🛟 자주 묻는 질문** 섹션을 먼저 확인. 봉주가 직접 따라하면서 막혔던 부분들이 모두 들어있어요.
